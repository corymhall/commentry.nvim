# Quickstart: Diff Line Comments

## Goal

Open a diff view of local changes, add draft comments to specific lines, and
manage those comments before submission (submission not included in this
release).

## Steps

0. Ensure `diffview.nvim` is installed and loaded.
1. Open a project with local changes.
2. Run the command to open the diff view for current changes.
3. Move the cursor to a target line in the diff.
4. Trigger the add-comment keymap to create a draft comment.
5. Edit or delete the draft comment while staying in the diff view.
6. Close and reopen the editor to confirm draft comments persist.

## Expected Outcome

- The diff view opens and shows local changes.
- Draft comments appear at the intended lines.
- Draft comments can be edited and deleted before submission.
- Draft comments persist across editor restarts.
